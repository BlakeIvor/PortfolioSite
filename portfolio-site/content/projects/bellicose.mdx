# Bellicose

**Technologies used:** C#, Unity, Git

![Bellicose](/static/images/bellicose/bellicose-title.png)

## Project Overview

A dynamic 2D fighting game built with Unity's new Input System, featuring local multiplayer combat, multiple unique characters, and sophisticated fighting mechanics with input buffering and state management. Bellicose delivers the fast-paced, precise combat that fighting game enthusiasts demand, emphasizing skill, timing, and strategy in every match.

## Key Features

- **Advanced Input System**: Leverages Unity's latest input framework for responsive, lag-free controls across various devices
- **Multi-Device Support**: Seamless support for various controllers and input devices with automatic configuration
- **Multiple Character Roster**: Unique characters with distinct movesets, abilities, and strategic approaches
- **Local Multiplayer Focus**: Optimized for local competitive play with zero-latency responsive controls
- **Frame-Perfect Combat**: Precise frame data implementation for competitive-level gameplay and tournament viability

## Technical Challenges

- **Input Buffering System**: Implementing advanced input buffering to ensure complex combos register reliably under pressure
- **Character State Machines**: Developing sophisticated state machines for fluid animation transitions and move execution
- **Combat Balance**: Carefully tuning character abilities, frame data, and mechanics for competitive balance
- **Performance Optimization**: Maintaining consistent 60fps gameplay with pixel-perfect collision detection
- **Controller Integration**: Ensuring responsive input handling across different controller types and configurations
- **Combo System Implementation**: Creating deep combo mechanics that reward skill and creativity while remaining accessible

Bellicose captures the essence of classic fighting games while incorporating modern design sensibilities. The game is accessible to newcomers while offering the depth that dedicated fighting game players crave, resulting in a polished experience that demonstrates mastery of Unity's systems and deep understanding of fighting game design principles.
